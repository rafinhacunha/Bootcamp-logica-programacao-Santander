# Fluxograma

## Descrição
Esse é um tópico para abordar o conceito de fluxograma e como isso pode ser importante para entender algoritmos e lógica de programação

## Definição
Fluxograma é uma maneira de registrar um algoritmo visualmente, muito importante para quem é iniciante na área e tem problemas em visualizar o problema e suas possíveis resoluções.

## Exemplo
Fluxogramas foram desenvolvidos durantes as aulas para alguns problemas, e também foi mostrados os diferentes tipos de simbolos e suas utilizações.
