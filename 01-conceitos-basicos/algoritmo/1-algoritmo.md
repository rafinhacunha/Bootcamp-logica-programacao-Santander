# Algoritmo

## Descrição
Esse é um tópico para abordar o conceito/definição de algoritmo, é importante para dar o primeiro passo em relação a **lógica de programação**.

## Definição
Algoritmo é o passo a passo lógico do que deve ser executado para resolver um determinado problema.

## Exemplo 
O problema: Quais ações nosso herói precisa realizar para passar dessa fase? 
Isso em um cenário onde existe o herói no canto esquerdo e logo a sua frente um adversário, e atrás do vilão um buraco, o "desafio" seria passar essa fase, utilizando os conceitos de algoritmo, os comandos são: andar para esquerda/direita, pular e atacar, como o algoritmo é uma sequência lógica a atividade é atribuir a sequência de comandos para passar essa fase, no caso seria ir para direita, logo após atacar o inimigo depois de eliminá-lo, continuar andando para direita e pular o buraco, assim formando uma sequência lógica que consiga finalizar o problema, e também utilizando o conceito de algoritmo para tudo isso.
