# Conceitos de Input, Process e Output

## Definição 
Input: São os comandos de entrada, é basicamente as informações que o usuário vai fornecer pro aplicativo, site e etc.
Podendo ser: login, senha, email, solicitamente de alguma ação e etc...
Process: São ações que serão executadas pelo algoritmo com os dados que o usúario forneceu ou não depende do aplicativo, para realizar certas ações.
Podendo ser: verificar se já existe alguem com esse mesmo login, verificar se o pagamento foi aprovado, e etc...
Output: São os comandos de saída, é um retorno de algo para o usuário, podendo ser uma mensagem ou algum posicionamente.
