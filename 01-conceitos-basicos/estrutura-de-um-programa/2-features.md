# Conceito de Features

## Definição
Features é basicamente uma seríe de funcionalidades que o aplicativo, site ou seja o que for, vai ter, por exemplo: um aplicativo de corrida como a Uber, existem diversas funcionalidades nela, essas sendo: solicitar veículo, cadastrar usuário, cadastrar motorista.
cada uma dessas funcionalidades são features, e essas features possuem todo o processo de algoritmo que foi citado anteriormente, com input, process e output.
